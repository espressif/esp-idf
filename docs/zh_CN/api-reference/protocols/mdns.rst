mDNS 服务
=========
:link_to_translation:`en:[English]`

mDNS 是一种组播 UDP 服务，用来提供本地网络服务和主机发现。

自 v5.0 版本起，ESP-IDF 组件 `mDNS` 已从 ESP-IDF 中迁出至独立的仓库：

* GitHub 上 `mDNS 组件 <https://github.com/espressif/esp-protocols/tree/master/components/mdns>`__
